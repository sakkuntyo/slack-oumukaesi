# slackでオウム返しするBot

## 動作環境

- ubuntu 18.04
- nodejs 8.11.0
- express framework

## 起動方法

```
# nodejsのインストール
$ git clone https://github.com/creationix/nvm ~/.nvm
$ source ~/.nvm/nvm.sh
$ echo "source ~/.nvm/nvm.sh" >> ~/.bashrc
$ nvm install 8.11.0
$ nvm use 8.11.0
# このアプリの起動
$ git clone https://github.com/sakkuntyo/slack-oumukaesi
$ cd slack-oumukaesi
$ npm install
$ npm start
```
